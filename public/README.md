# CXRNER MUSIC Mini App

Полностью русская Telegram Mini App версия для лейбла:

- тёмный минималистичный UI + glow/анимации;
- разделы: Главная, Релизы, Артисты, Анкета, Контакты;
- анкета отгрузки повторяет ключевые вопросы из бота;
- отправка идёт через `Telegram.WebApp.sendData()` в `main.py`.

## Структура

- `index.html` — разметка Mini App
- `styles.css` — дизайн, анимации, адаптив
- `app.js` — логика интерфейса и отправка анкеты
- `assets/logo.svg` — лого CXRNER для загрузчика и шапки

## Локальный запуск

```bash
python -m http.server 8080 -d webapp
```

После запуска: `http://localhost:8080`.

## Подключение к боту

В `main.py` должен быть задан:

```bash
WEBAPP_URL=https://ваш-домен/index.html
```

Обработчик в боте принимает action:

- `webapp_release_submit` (актуальный)
- `submit_release` (legacy fallback)

После приёма анкеты бот:

- валидирует данные;
- отправляет анкету в модерацию;
- добавляет релиз в `releases.json` (раздел «Мои релизы» у артиста);
- использует тот же функционал статусов/отклонения/UPC, что и у анкеты из бота.
